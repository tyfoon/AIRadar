Consistentie-regels:

  • Duration chips: zelfde stijl, zelfde set (1h/4h/8h/24h/Custom.../Permanent), zelfde positie — overal
  • "Permanent" altijd rechts, gescheiden door een vertical divider
  • Hover: blue-500/15 + blue-300 tekst — voor zowel alert als activity
  • Selected toggle (scope/action): bg-blue-600 text-white shadow-sm (blue = toggle state)
  • Selected duration chip: bg-white/[0.10] text-slate-200 font-semibold (NOT blue — blue reserved for toggles)
  • Card body: altijd single row (icon + content + time + actions rechts)
  • Tabs alleen bij service-alerts (waar Activity Control beschikbaar is)
  • Dismissed op detail-pagina's: trash icoon voor permanent verwijderen
