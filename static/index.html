Setup Guide

Install CrowdSec to enable Active Protect:

$ curl -s https://install.crowdsec.net | sudo bash

$ sudo apt install crowdsec crowdsec-firewall-bouncer-iptables

$ sudo cscli collections install crowdsecurity/linux

Once installed, CrowdSec will automatically connect to the community threat intelligence network and begin protecting your network.

CrowdSec Engine

Community-driven threat intelligence

Automatically blocks known malicious IPs, botnets, and hackers using CrowdSec's real-time threat database.

0 threats blocked

How it works

  • 1. CrowdSec analyses Zeek network logs for suspicious patterns
  • 2. Detected attackers are matched against a global reputation database
  • 3. Confirmed threats are automatically blocked at the network level
  • 4. Your network contributes back to the community intelligence

Threat Intelligence Sources

Botnet Detection

Blocks traffic from known botnets, command-and-control servers, and zombie networks.

Brute Force Protection

Detects and blocks automated password-guessing attacks on SSH, web apps, and other services.

Scanner & Exploit

Identifies port scanners, vulnerability probes, and known exploit attempts targeting your network.
