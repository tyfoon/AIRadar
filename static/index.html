How Access Control Works

Global filters use AdGuard Home's built-in category blocking to instantly block entire classes of services (social media, gaming, adult content) network-wide.
Granular controls add DNS-level rules (||domain^) to selectively block individual AI or Cloud services. Temporary blocks auto-expire after the set duration.
All blocking is enforced at the DNS level — devices must use 127.0.0.1 as their DNS server. Open AdGuard Dashboard →
